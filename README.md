# テーマ

ナチュラルなテイストで全体をまとめ、統一感のあるデザインを意識しています

# 機能

## 基本ページ

- インデックス
- 作品集（ギャラリー）
- プロフィール
- 質問＆回答（制作者に聞きたいこと）
- お問い合わせ［新規］

## サブページ

- 作品情報
- PDF閲覧（ポートフォリオ PDF）［新規］

## 独立機能［新規］

- 猫違いゲーム
- ポスター紹介ページ
- ハンドクリームセールスページ

## 全体機能

- トップに戻る猫
- サイドメニュー（モバイル）［新規］
